Soccer Management Game

Github

A prototype soccer management game built using VueJS. The game is played completely on the client side using IndexedDB to store data. A user can choose a country to play as and then sign players, simulate games, compete in tournaments, and win championships.

Javascript, VueJS, Semantic-UI

Efed Content Management System

Github

A content management system built with the "e-fed" in mind. Efed standing for "Electronic Federation" is a fictitious online pro wrestling organization. Using the software the owner can sign wrestlers, book events, and customize the look and feel of the federation. The wrestlers can create roleplays for the events and also converse through forums and the private messaging system. Other features include: custom pages, wrestler bios, title management, and the ability to add staff members. Use staff as username and password to login.

PHP, Laravel, Bootstrap

Timesheet Management System

Github

The timesheet management system allows users to post their work time for a specific date. Regular and overtime hours are recorded. Each user that is created has an associated regular and overtime rate which is used further by the admin. An admin can create new users, update employee time, and update employee rates. An admin has a calendar overview of every employee to see the dates and times worked. Finally the admin can generate reports for all users or a specific user to calculate overall time worked. Use admin as username and password to login.

PHP, Laravel, Jquery, Bootstrap

ePractical Taxes

Epractical Taxes is an online program for users to fill out their information for taxes in order for it to be prepared by a tax preparer. Users sign up and go through an eleven step process of answering tax related questions. This information is then compiled for the tax preparer for easy entry into the tax software (not related to online interface). Once the tax return is prepared it is uploaded up back to the user and then the user is notified via email to download their return.

PHP, Laravel, Jquery, Bootstrap

EfedZone

Efedzone is an efedding community. EfedZone has the efed content management system built into it. Efedzone comprises of multiple feds instead of a just a single fed with users being able to create multiple wrestlers. Wrestlers can sign with multiple federations. Efedzone features community forums, a site wide private messaging system, and federation news that comes from the individual feds.

PHP, Laravel
